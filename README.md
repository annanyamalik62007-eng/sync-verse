# Syncverse

**The operating system for university life.**

Syncverse connects verified students around the world based on intent — what they want to accomplish *right now* — instead of static profiles, follower counts, or endless swiping. It's built to remove the friction in finding collaborators, mentors, teammates, and opportunities.

**Live demo:** https://campus-connect-engine.replit.app/

## Why it exists

Students struggle to build meaningful relationships across campuses. Most existing platforms optimize for content and attention rather than purposeful collaboration. Syncverse flips that: it's built around outcomes, not attention — collaboration before content, opportunity over popularity.

## Core principles

- **Verified identities** — every user signs up with a `.edu` university email
- **AI-first matching** — an intent engine, not a static profile browser
- **Trust and safety** — moderation, reporting, and collaboration ratings built in
- **Global yet campus-centric** — designed to start hyperlocal and scale outward

## How matching works

Instead of a static profile, users declare an active objective — what they're trying to get done today (find a teammate, a mentor, a co-founder, a study group). The matching engine scores compatibility using skills, availability, location, goals, past collaborations, and mutual interests, and recommendations update dynamically as intent changes.

**Example:** a student posts "Need a UI designer for a climate-tech hackathon" → gets ranked matches → opens a project workspace → messages, schedules, and ships the project — all inside the platform.
